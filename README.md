# 🤖 Productivity Bot

Многофункциональный Telegram бот для управления задачами, расходами и временем. Поддерживает 3 языка и 4 валюты.

## 🌟 Возможности

### 📋 Задачи
- Создание задач с названием, описанием, дедлайном
- Выбор приоритета (высокий/средний/низкий)
- Категории задач
- Отметка выполненных задач

### 💰 Расходы
- Добавление расходов с выбором валюты (RUB/USD/EUR/UAH)
- Категории расходов
- Статистика за день/неделю/месяц
- Просмотр по категориям

### ⏰ Таймер
- Таймер для работы по методу Pomodoro
- Пауза и остановка
- Статистика сессий

### 🌐 Мультиязычность
- Русский
- English
- Українська
